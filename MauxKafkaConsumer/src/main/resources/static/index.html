Mauxilium Kafka Consumer Agent


This is the Consumer entry page of a simple Producer/Consumer communication through Kafka Broker.
Different scenarios could be tested by the way of Docker-Compose scripts in the project Docker directory.
Here you have nothing to do but:

  1. Refer to the Producer(s) page (es: http://localhost:8081) in order to run the tests.
  2. Go to Kibana interface (http://localhost:5601/) in order to verify any communication results.
